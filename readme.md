Szablon
=======
Dla prezentacji w impress.js, zawiera:

1. wpięte impress w HTMLa z kilkoma punktami
2. podstawowy CSS z wyróżnieniem, cytatem, podkreślonym krokiem i obsługą paru transformat impressa.
3. kilka pomocnych grafik.

Na żywo:
